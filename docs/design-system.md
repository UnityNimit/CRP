# Campus Recruiting Portal — Design System

## Brand

**Campus Placement Portal** — institutional career platform aesthetic.

## Color tokens

| Token | Value | Use |
|-------|-------|-----|
| --color-ink | #0B1F33 | Primary text |
| --color-surface | #F5F7FA | App background |
| --color-panel | #FFFFFF | Panels |
| --color-accent | #0E7C7B | CTA, links |
| --color-accent-hover | #0A5F5E | Hover |
| --color-border | #D7DEE7 | Borders |
| --color-success | #1B7F4E | Approved, selected |
| --color-warning | #B7791F | Pending |
| --color-danger | #B42318 | Rejected |

## Typography

- Display: Fraunces / Source Serif 4
- Body: IBM Plex Sans

## Layout

- Left nav + top bar shell per role
- Tables for dense data; cards only for interactive units
- Status chips for all workflow states

## Motion

- Route fade-slide ~180ms
- Status chip transitions
- Chart draw-in on analytics load

## Status labels

| Status | Label |
|--------|-------|
| PENDING | Awaiting approval |
| APPROVED | Live |
| REJECTED | Rejected |
| CLOSED | Closed |
